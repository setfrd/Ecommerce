# Grocery Store Project

# RAF & CO.

A full-stack MERN e-commerce web application.

## Stack

- Frontend: React (Create React App), React Router
- Backend: Node.js, Express, Mongoose
- Database: MongoDB Atlas
- File uploads: Multer

## Setup

Backend:

1. cd backend
2. npm install
3. create a .env file with MONGO_URI and PORT
4. node index.js

Frontend:

1. cd frontend
2. npm install
3. npm start
